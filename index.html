About Me

My Skills

I’m a dedicated frontend web developer with solid experience in building fast, responsive, and interactive web applications using today’s most powerful tools. My core strength lies in Tailwind CSS, which I use to design clean, responsive user interfaces with precision and efficiency. It helps me build visually appealing layouts that are easy to maintain and scale. For dynamic and interactive functionality, I work primarily with JavaScript and React.js. React allows me to build reusable components and manage application state effectively, ensuring smooth and efficient user experiences. Expanding beyond the frontend, I also work with Node.js and Next.js to develop full-stack applications. Next.js enables server-side rendering and better performance, while Node.js handles backend operations and APIs. For database management, I use MongoDB, a flexible NoSQL database that supports scalable data structures. I also rely on Firebase for authentication, hosting, and real-time database features, which help me deploy and manage apps quickly. I enjoy turning ideas into functional, modern web solutions. My focus is on writing clean code, following best practices, and continuously learning new technologies to stay ahead in web development.
